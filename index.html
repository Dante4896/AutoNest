GitHub quick guide

What to do after downloading this file

Create a new public GitHub repository.
Rename this file to index.html if it is not already.
Upload the file into the repository root.
Go to Settings → Pages.
Select Deploy from a branch, choose main and /(root).
Save and wait for the live URL.

FAQ

A few things you may ask next.

This setup is meant for easy launching, not for a full custom e-commerce backend.

Can GitHub Pages take payments directly?

No. Use GitHub Pages for the website itself, then send buyers to Gumroad, Payhip, or another payment platform for checkout and file delivery.

Can I add more bundles later?

Yes. Duplicate any product card section, change the title, price, description, and button link, then upload the edited file again.

Can I use this on mobile?

Yes. This page is responsive and built to stack into a single-column layout on smaller screens.
